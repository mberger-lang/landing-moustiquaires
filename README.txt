Pour lancer le site en local :
1. Ouvrir un terminal dans ce dossier
2. npm install
3. npm run dev

Les images actuelles sont des visuels de remplacement en SVG pour que le site
fonctionne tout de suite. Si tu veux remettre tes vraies images, remplace dans
public\images les fichiers suivants :
- logo-caseo.svg
- showroom-caseo-chambery.svg
- magasin-caseo-chambery.svg
- moustiquaire-magasin-fenetre.svg
- moustiquaire-magasin-baie-vitree.svg

Si tu veux utiliser des JPG ou PNG a la place, garde les noms ou modifie les
extensions correspondantes dans pages\index.js.
